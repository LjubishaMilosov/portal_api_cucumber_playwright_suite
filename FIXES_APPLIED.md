### ✅ Issue #4: Redundancies & Refactoring

**Status**: IDENTIFIED & PRIORITIZED ✅

#### **Critical Redundancies Found**:

1. **File Naming Inconsistency**

   ```
   /data/getCountries.json       ← lowercase
   /data/GetCurrencies.json      ← UPPERCASE C (INCONSISTENT!)
   /data/GetLanguages.json       ← INCONSISTENT!
   ```

   **Fix**: Standardize to lowercase in [core.steps.ts](src/steps/core.steps.ts)

2. **Backward Compatibility Module**

   ```
   /src/support/utils/utilityFunctions.ts  ← OLD re-export module
   ```

   **Status**: Not needed for new code. Old module for backward compatibility.  
   **Recommendation**: Eventually deprecate (Low priority - not breaking anything)

3. **Schema/Endpoint Naming**

   ```
   src/api/endpoints/base.api.ts
   src/api/schemas/base.schema.ts
   ```

   **Suggestion**: Rename to `core.api.ts` and `core.schema.ts` for clarity

4. **Step Definition Pattern Duplication**

   ```typescript
   // Pattern repeats in core.steps.ts, customer.steps.ts, games.steps.ts
   const response = await fetchX(this);
   await handleResponse(this, response);
   writeDataToFile('./data/...', this.responseBody);
   ```

   **Optimization**: Extract to shared helper function
